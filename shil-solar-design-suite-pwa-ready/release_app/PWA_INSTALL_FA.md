# نصب اپ تحت وب SHIL Solar

این نسخه به صورت PWA آماده شده و روی ویندوز، اندروید و آیفون قابل نصب است.

## ویندوز - Chrome یا Edge
1. لینک اپ را باز کنید.
2. کنار نوار آدرس روی آیکون Install / نصب کلیک کنید.
3. گزینه Install را بزنید.
4. اپ مثل نرم‌افزار مستقل باز می‌شود.

## اندروید - Chrome
1. لینک اپ را در Chrome باز کنید.
2. منوی سه‌نقطه را بزنید.
3. گزینه Add to Home screen یا Install app را انتخاب کنید.
4. تایید کنید.

## آیفون / آیپد - Safari
1. لینک اپ را فقط با Safari باز کنید.
2. دکمه Share را بزنید.
3. گزینه Add to Home Screen را انتخاب کنید.
4. Add را بزنید.

## نکات مهم
- برای نصب روی iPhone باید از Safari استفاده شود.
- اولین بار باید اینترنت باشد؛ بعد از بارگذاری اولیه، بخش‌های اصلی اپ به صورت آفلاین هم باز می‌شوند.
- اگر آپدیت جدید منتشر شد، یک بار اپ را کامل ببندید و دوباره باز کنید.
