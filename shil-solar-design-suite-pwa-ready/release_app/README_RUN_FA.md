# راه‌اندازی نهایی Solar Design Suite

## وضعیت پروژه
این پروژه build شده و خروجی production داخل پوشه `dist/` آماده است.

## اجرای پروژه روی سیستم (حالت توسعه)
پیش‌نیاز:
- Node.js 18 یا بالاتر
- npm

دستورها:
```bash
npm ci
npm run dev
```

بعد در مرورگر باز کن:
```text
http://localhost:5173
```

## اجرای نسخه نهایی روی سیستم
اگر فقط بخواهی نسخه نهایی را اجرا کنی:

### روش ساده با Python
```bash
cd dist
python -m http.server 8080
```

بعد باز کن:
```text
http://localhost:8080
```

### روش با npm package
می‌توانی از هر static server هم استفاده کنی، مثل:
```bash
npx serve dist
```

## اجرا روی گوشی
برای تست روی گوشی در شبکه داخلی:

### روش 1: با dev server
```bash
npm run dev -- --host
```

بعد IP سیستم خودت را پیدا کن و در گوشی باز کن:
```text
http://YOUR_LOCAL_IP:5173
```
مثال:
```text
http://192.168.1.10:5173
```

### روش 2: با نسخه نهایی build شده
```bash
cd dist
python -m http.server 8080
```
و در گوشی باز کن:
```text
http://YOUR_LOCAL_IP:8080
```

## نصب روی گوشی به صورت PWA
وقتی سایت را در Chrome یا Edge موبایل باز کردی:
- منوی مرورگر
- Add to Home Screen / Install App

بعد اپ مثل یک برنامه مستقل روی گوشی نصب می‌شود.

## فایل‌های مهم
- `src/` سورس نهایی پروژه
- `dist/` خروجی build نهایی
- `public/` فایل‌های پابلیک و دارایی‌های تصویری

## قابلیت‌های فعلی
- طراحی Backup / Off-Grid / Hybrid / Grid-Tie
- روش‌های مختلف محاسبه
- پروفایل مصرف ساعتی
- شبیه‌سازی SOC و انرژی
- ذخیره پروژه و نسخه‌بندی
- بانک تجهیزات + تجهیزات سفارشی
- گزارش PDF
- PWA برای اجرا روی گوشی

## نکته مهم
داده‌های پروژه در مرورگر کاربر ذخیره می‌شوند. اگر خواستی در مرحله بعدی می‌توانیم:
- خروجی و ورودی را به فایل JSON صادر/وارد کنیم
- یا ذخیره ابری/SQLite/PostgreSQL اضافه کنیم
